# Sistema-Gestion-Inventarios-Java
Proyecto Final - Sistema de Gestión de Inventarios desarrollado en Java utilizando Programación Orientada a Objetos. Implementa herencia, polimorfismo, clases abstractas e interfaces.

# Sistema de Gestión de Inventarios

## Descripción

Este proyecto fue desarrollado en Java como parte de la Actividad Formativa 4.

El sistema permite administrar diferentes tipos de productos utilizando los principios de la Programación Orientada a Objetos.

## Características

- Clases abstractas
- Interfaces
- Herencia
- Polimorfismo
- Sistema de inventario

## Clases

- Producto (Clase Abstracta)
- Electronico
- Alimento
- Ropa
- Vendible (Interface)
- Main
